## TESTING METHODOLOGY
The main runs the test of the following matrix expression:

D = A + (# * B) * C

Where:

A = [6, 4
     8, 3]

B = [1, 2, 3
     4, 5, 6]

C = [2, 4, 6
     1, 3, 5]

The output of D is then printed to the console for verification
